# Спецификация: Обновление выгрузки Алушта/ФРП
**Версия:** 1.0.0
**Статус:** Выполнено

## 1. Описание задачи
Адаптация сводных отчетов под требования "Алушта/ФРП".

## 2. Требования
### 2.1 Интерфейс
- Переименовать кнопку `excelReport` на ленте в "Выгрузка Алушта/ФРП".

### 2.2 Логика данных
- В листе "Штат" найти столбец "Лицо", содержащий цифровые значения (Табельный номер). Игнорировать текстовое поле "Лицо" (ФИО).
- В отчетах (ДСО и Риск) добавить столбец "Табельный номер" после "Личного номера".
