# Спецификация: Самовосстанавливающаяся архитектура (Self-Healing)
Описание
Система автоматического восстановления удаленных листов и стертых заголовков для защиты от человеческого фактора.

Требования
Макрос должен уметь проверять наличие листов "ДСО" и "Надбавки без периодов".

Если лист удален, скрипт воссоздает его в конце книги.

Скрипт алгоритмически генерирует заголовки:

Для "ДСО": базовая часть + цикл для 24 периодов (Начало1, Конец1 ... Начало24, Конец24).

Для "Надбавки без периодов": статический массив из 6 колонок.

Скрипт автоматически применяет UI-форматирование: жирный шрифт, заливка, автоширина, автофильтр и закрепление первой строки (Freeze Panes).
