# Спецификация: Гибридная система лицензирования (Time-Bomb + Admin Panel)
## Описание
Система оффлайн-защиты с трехуровневой проверкой даты истечения.

## Требования
- До даты PUBLIC_EXPIRY_DATE (вшита в код) программа работает без активации.

- После истечения требуется персональный ключ (привязка к HWID) или административная установка даты.

- Скрытая панель администратора в frmAbout (доступ по паролю) для установки индивидуального срока файла перед отправкой.

- Защита от перевода системных часов.
