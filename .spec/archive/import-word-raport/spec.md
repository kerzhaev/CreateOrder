# Спецификация: Импорт утвержденных рапортов (Word) в ДСО

**Версия:** 1.0.0
**Ветка:** `feature/import-word-raport`
**Статус:** Draft
**Автор:** Кержаев Евгений

## 1. Цель (User Stories)
**US1 (Главный сценарий):** Как пользователь, я хочу нажать кнопку на ленте Excel, выбрать утвержденный рапорт в формате Word, чтобы система автоматически извлекла из него периоды участия в СВО и добавила их в лист "ДСО" к соответствующим сотрудникам.
**US2 (Агрегация):** Как пользователь, я ожидаю, что если в рапорте один сотрудник встречается несколько раз (несколько периодов), скрипт сгруппирует эти периоды и запишет их в одну строку на листе "ДСО" последовательно слева направо.
**US3 (Отказоустойчивость):** Как пользователь, я хочу, чтобы макрос игнорировал пустые строки, шапку таблицы и не падал при опечатках в датах (кривые даты должны быть загружены как текст для последующей подсветки модулем `mdlDataValidation`).

## 2. Требования (Requirements)
* **FR-001:** Система должна конвертировать Word-документ во временный HTML-файл для надежного парсинга таблиц средствами Excel.
* **FR-002:** Поиск и сопоставление сотрудников между рапортом и листом "ДСО" производится строго по колонке "Личный номер".
* **FR-003:** Если сотрудник из рапорта не найден в листе "ДСО", система должна создать для него новую строку в конце списка, подтянув актуальное Звание и Должность из листа "Штат".
* **FR-004:** Временные файлы (HTML) и скрытые экземпляры приложения Word должны гарантированно уничтожаться/закрываться после завершения работы (или при ошибке).
* **FR-005:** По завершении операции должен выводиться отчет: сколько строк прочитано, сколько периодов добавлено, сколько создано новых сотрудников.
