# Fix Encoding Issue - Спецификация задачи

## Описание проблемы
При работе на системе Windows с английской локализацией (English system locale) VBA код выдаёт ошибки, потому что не может найти листы с русскими именами через строковые обращения `ThisWorkbook.Sheets("ДСО")`.

Проблема связана с кодировкой символов при сохранении и чтении VBA кода на разных системах.

## Требуемое решение
Перейти от использования строковых имён листов к кодовым именам (code names), которые не зависят от кодировки системы.

**До (проблема):**
```vba
Set wsMain = ThisWorkbook.Sheets("ДСО")
Set wsStaff = ThisWorkbook.Sheets("Штат")
```

**После (решение):**
```vba
Set wsMain = wsDSO
Set wsStaff = wsStaff
```

## Затрагиваемые листы
| Отображаемое имя | Кодовое имя (до) | Кодовое имя (после) |
|------------------|------------------|---------------------|
| ДСО              | Лист1 (пример)   | `wsDSO`             |
| Штат             | Лист2 (пример)   | `wsStaff`           |
| Выплаты_Без_Периодов | -         | `wsPayments`         |
| Справочник_ВУС_Экипаж | -         | `wsRefVUSCrew`       |
| Справочник_Типы_Выплат | -         | `wsRefPaymentTypes`  |

## Затрагиваемые модули
Следующие модули используют строковые имена листов:
1. `mdlHelper.bas` -多处使用 "Штат" и "ДСО"
2. `mdlMainExport.bas` - использует "ДСО" и "Штат"
3. `mdlRaportExport.bas` - использует "ДСО" и "Штат"
4. `mdlSpravkaExport.bas` - использует "ДСО" и "Штат"
5. `mdlRiskExport.bas` - использует "ДСО" и "Штат"
6. `mdlFRPExport.bas` - использует "ДСО" и "Штат"
7. `mdlRibbonHandlers.bas` - использует "ДСО", "Штат", и справочники
8. `mdlReferenceData.bas` - использует константы с русскими именами
9. `frmSelectEmployee.frm` - использует "Штат"
10. `frmSearchFIO.frm` - использует "Штат" и "ДСО"

## Примечания
- Кодовые имена листов (Name в скобках) нужно будет изменить вручную в Excel/VBA редакторе
- Отображаемые имена вкладок (Name без скобок) остаются на кириллице для пользователя
- Константы в `mdlReferenceData.bas` можно оставить, но лучше заменить на кодовые имена для единообразия
